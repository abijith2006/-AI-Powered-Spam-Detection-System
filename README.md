🛡️ AI-Powered Spam Detection System
🚀 Overview

AI-Powered Spam Detection System is a production-ready machine learning application designed to detect spam messages and phishing URLs in real-time. Built with Python and Scikit-learn, it uses a Random Forest classifier to analyze content and provide accurate predictions along with confidence scores.

The system features a modern dark-themed dashboard, real-time analytics, scan history management, and a lightweight architecture that runs entirely on your local machine without requiring external servers.
